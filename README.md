# 🤖 AI-Powered Credit Delinquency Risk Analysis  
### 🏢 Tata iQ Job Simulation – Forage

## 📌 Overview
This repository documents my work from the **Tata iQ Job Simulation** on the **Forage** platform, 
where I simulated the role of a **data analytics consultant** supporting a financial services client.

The objective of the project was to design an **AI-powered approach** to identify, predict, 
and reduce customer credit delinquency using **data analytics**, **Generative AI tools**, 
and **responsible AI principles**.

----

## 🧩 Business Problem
Financial institutions face increasing challenges in managing **customer credit delinquency**, 
which directly impacts **revenue**, **regulatory compliance**, and **customer experience**.

The key goals of this simulation were to:
- 🔍 Identify early-stage delinquency risk indicators  
- 📊 Structure insights from exploratory data analysis (EDA)  
- 🧠 Propose a no-code predictive modeling framework  
- ⚙️ Design an AI-driven collections strategy  

----

## 👤 My Role
During this simulation, I:
- Conducted exploratory data analysis (EDA) using **GenAI-assisted workflows**
- Assessed data quality and identified potential **risk indicators**
- Designed a **no-code predictive modeling framework**
- Proposed an **ethical, scalable AI-driven collections strategy**
